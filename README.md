# 🎬 Netflix Data Analysis Dashboard (Tableau)

## 📌 Overview
This project presents an interactive **Netflix Data Analysis Dashboard** built using **Tableau**. It helps analyze Netflix’s content library and uncover insights related to movies, TV shows, genres, countries, ratings, and release trends.

The aim of this project is to convert raw data into meaningful visual insights for better understanding of content distribution and trends on streaming platforms.

---

## 📊 Key Insights

- 📺 Distribution of Movies vs TV Shows  
- 🌍 Country-wise content availability  
- 🎭 Most popular genres on Netflix  
- 📅 Year-wise trend of content addition  
- ⭐ Ratings distribution (TV-MA, PG, etc.)  
- 🔎 Interactive filters for dynamic analysis  

---

## 🛠️ Tech Stack

- **Tableau Desktop** – Data visualization  
- **Dataset** – Netflix dataset (CSV format)  
- **GitHub** – Version control and hosting
